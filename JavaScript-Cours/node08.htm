      (* traitements *)

      // Traitements
}

Structure Répéter … Jusqu'à

Il n'existe pas d'équivalent pour la structure Répéter … Jusqu'à en JavaScript.
Il est, cependant, possible de simuler la structure Répéter … Jusqu'à de la manière suivante :
