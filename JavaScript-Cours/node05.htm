Les opérateurs

Les opérateurs sont classées dans JavaScript en cinq catégories :

Les opérateurs de calcul

Addition Soustraction Multiplication Division Modulo
a=3+2; b=3-5; c=2*9; d=1/8; e=5%2;

Les opérateurs de comparaison

égal inférieur supérieur différent inférieur ou égal supérieur ou égal
a==b a<b a>b a != b a<=b a>=b

Les opérateurs associatifs

a = a + 5; b = b * 2; c = c – 3; d = d / 4;
a += 5; a *= 2; c -= 3; d /= 4;

Les opérateurs logiques

Et Ou Non
a==5 && b==3 a==5 || b==3 !(a == b)

Les opérateurs d'incrémentation

a = a + 1; a = a + 1; a = a - 1; a = a - 1;
a++; ++a; a--; --a;
