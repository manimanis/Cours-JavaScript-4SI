Les Entrées/Sorties

Entrée des données

L'utilisateur peut introduire des données dans un script JavaScript en utilisant une des méthodes suivantes :

Formulaire HTML

var nom, prenom;
nom = document.f.nom.value;
prenom = document.f.prenom.value;
Boîte d'invite

age = prompt("Votre age SVP ? ", "17");

La variable age contient :

  • La saisie de l'utilisateur si ce dernier appuie sur [OK].
  • null : si ce dernier appuie sur [Annuler]
Boîte de confirmation

rep = confirm("Voulez-vous vraiment ouvrir ce lien");

La variable rep contient :

  • true : si l'utilisateur appuie sur [OK].
  • false : si ce dernier appuie sur [Annuler]

Affichage des résultats

Les résultats d'exécution des scripts peuvent être visualisés de trois façons et ce à travers :

Formulaire HTML

document.f.nom.value = "MANI";
document.f.prenom.value = "Mohamed Anis";
Boîte d'alerte

a = 5; a2 = a * a;
alert("Le carré de " + a + " est " + a2);
Utilisation du document

a = 5; a2 = a * a;
document.write("Le carré de " + a + " est " + a2);
