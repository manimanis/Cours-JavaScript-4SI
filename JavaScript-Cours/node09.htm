Les fonctions

Une fonction est déclarée en JavaScript de la façon suivante :

function nom_fonction(param1, param2, …, paramN) {
         // traitement
         return valeur; // valeur retournée par la fonction
}

Pour JavaScript une procédure est une fonction qui ne retourne aucune valeur ou tout simplement c'est une fonction où le mot clé return est omis.

Exemple :

function f(x) { return 2*x+8; }
