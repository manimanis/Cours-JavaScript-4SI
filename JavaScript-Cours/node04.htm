Les variables

Il est possible de déclarer une variable en JavaScript de deux façons :

  • Explicitement : en utilisant le mot clé var suivi par le nom de la variable.
  • Implicitement : en omettant le mot clé var tout en affectant une valeur au nom de la variable.

Délaration Explicite

En utilisant le mot clé var.

var a, b, c = 3;
a = "Bonjour";
b = 15 * c; // b = 45

Délaration Implicite

En affectant directement un valeur à un nom de variable.

c = 3;
a = "Bonjour";
b = 15 * c; // b = 45

JavaScript est sensible à la casse

Remarque : JavaScript est sensible à la casse.

L'exemple suivant démontre que JavaScript différencie la variable a (minuscule) de la variable A (majuscule).

A = 5; a = 25;
document.write("a= " + a + " - A=" + A);

Remarque :
Les variables déclarées au sein d'une fonction avec le mot clé var sont considérées comme étant des variables locales.
Les variables déclarées sans le mot clé var sont des variables globales quelque soit leurs emplacements.

Types de variables

JavaScript supporte quatre types de variables prédéfinies :

  • Les nombres Number (entiers/réels).
  • Les booléens (true ou false).
  • Les chaînes de caractères String ("Ahmed" ou 'Ahmed').
  • Les types Object (Objet).

Les variables vides contiennent la valeur : null.
