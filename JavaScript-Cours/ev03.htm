Fonctionnement

Lorsque le champ de texte np prend le focus (onFocus) le texte Saisissez votre nom et prénom s'affiche dans le champ status.

Lorsque le champ de texte np perd le focus (onBlur) le texte ... s'affiche dans le champ status.

Lorsque l'utilisateur relâche une touche du clavier (onKeyUp) le nombre de caractères tapés est mis à jour.

Lorsqu'on effectue un changement dans le champ de texte np et que l'on quitte ce champ (onChange) son contenu est transformé en majuscules.
