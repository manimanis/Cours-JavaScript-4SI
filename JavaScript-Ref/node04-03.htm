String

Fonction substring(...), substr(...)

Syntaxe

chaine.substr(deb)
chaine.substr(deb, long)

chaine.substring(deb)
chaine.substring(deb, fin)

Exemple

<script type="text/javascript">

var str="Hello world!";
document.write(str.substr(3)+"<br />");
document.write(str.substr(3,4));

document.write(str.substring(3)+"<br />");
document.write(str.substring(3,7));

</script>
